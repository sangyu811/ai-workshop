# AI内容工坊 - 零成本创业启动包

## 你拿到了什么

`startup-kit/` 目录下有两样东西：

| 文件 | 用途 |
|------|------|
| `index.html` | **接单展示页** - 打开就是你的服务官网 |
| `gen.py` | 生成器脚本（你已经用不上了，留着以后修改用） |

## 3 步启动

### 第 1 步：改微信 ID
打开 `index.html`，找到末尾 footer 部分：
```
微信: your_wechat_id_here
```
改成你的真实微信号。

### 第 2 步：发出去接单
**方式 A - 本地直接用**
双击 `index.html` 在浏览器打开，截图发朋友圈/小红书/闲鱼。

**方式 B - 挂到网上（推荐）**
1. 注册 [Vercel](https://vercel.com) 或 [Netlify](https://netlify.com)（免费）
2. 直接把 `startup-kit` 文件夹拖进去部署
3. 得到一个公网链接，可以放到闲鱼/小红书简介里

**方式 C - 最懒**
用 [Github Pages](https://pages.github.com)：
1. 新建 repo，把 `index.html` 传上去
2. 打开 Settings > Pages，选 main 分支
3. 等 2 分钟，就能访问了

### 第 3 步：去找客户
把链接发到这些地方：
- **闲鱼**：搜"代写""PPT""文案"，看别人怎么发，仿着发
- **小红书**：发笔记"帮大学生改简历/写文案"，挂上链接
- **豆瓣小组**：副业/文案/PPT 相关小组
- **朋友圈**：直接发，告诉朋友你开始接单了
